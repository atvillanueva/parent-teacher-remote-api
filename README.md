# parent-teacher-remote-api

1. git clone https://github.com/atvillanueva/parent-teacher-remote-api.git
2. parent-teacher-remote-api
3. sudo touch .env
4. sudo vi .env
5. copy and paste

```
   PORT=3000
   DATABASE_URL=mysql://root:mypassword@localhost:3306/mydb
```

6. sudo wq
7. npm install
8. npm run db:migrate
9. npm run db:push
10. npm start
